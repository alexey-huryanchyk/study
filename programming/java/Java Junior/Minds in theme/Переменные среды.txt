Переменная среды - текстовая переменная операционной системы, хранящая какую-либо информацию, например, данные о настройках системы.

echo %TMP%   ->   C:\Users\guru13\AppData\Local\Temp   - вернет путь к временной папке. TMP - имя переменной среды

Домашний каталог - это личный каталог пользователя в операционной системе, где находятся его данные, настройки и т.д.
C:\Users\guru13  - Windows
/home - Linux
/home/guru13 - UNIX

Для того, чтобы получить абсолютный путь к домашнему каталогу, необходим использовать стандартную переменную окружения:
HOMEPATH   ---> Windows
HOME   ----> MacOS, Linux

Чтобы получить значения переменной среды в Java  -  System.getenv("HOMEPATH")
Чтобы получить путь домашнего каталога в Java - System.getProperty("user.home")

PATH  - переменная, в которой указывается список всех папок, в которых надо искать исполняемый файл на выполнение
JAVA_HOME - переменная, в которой хранится путь к JDK, установленной на компьютере

Переменная окружения может быть задана:

- На уровне процесса интерпретатора командной строки или терминала. В этом случае такая переменная существует до тех пор, пока процесс запущен на операционной системе.

- На уровне текущего залогиненного пользователя. В этом случае такая переменная доступна ВСЕМ программам на текущем компьютере, которые запускаются от имени текущего залогиненного пользователя.
 Если на текущий компьютер залогинется другой пользователь, используя другую учетную запись, то такая переменная не будет доступна программам, которые запускаются от имени другого пользователя.

- На уровне системы. В этом случае такая переменная доступна ВСЕМ программам, которые запускаются от имени ВСЕХ пользователей, которые определены на текущем компьютере.
